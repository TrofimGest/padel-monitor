# Конкуренты: падел и ракеточные клубы Минска

Справочник действующих падел-клубов Минска. Используется судьёй для оценки
локации кандидата: далеко от конкурентов при хорошей локации =
недообслуженный район (плюс к спросу); вплотную к сильному клубу = риск
(но и кластерный эффект бывает — не запрет).

**Дополняй руками** по мере появления новых клубов. Координаты нужны для
расчёта расстояния (`padel_monitor/geo.py::nearest_competitor`); объекты без
координат в расчёте игнорируются, но остаются в списке как справка.

Данные читаются из YAML-блока ниже (имя, адрес, координаты, число кортов,
источник/точность координат). Первичное наполнение 2026-07-05.

```yaml
competitors:
  - name: "Padel Park"
    address: "пер. Софьи Ковалевской, 62 (р-н Нововиленская)"
    lat: 53.93709
    lon: 27.52883
    courts: null
    coords: osm                 # подтверждено OpenStreetMap
  - name: "Padel Club Minsk"
    address: "ул. Саперов, 5"
    lat: 53.9060
    lon: 27.5780
    courts: 2
    coords: approx              # координаты приблизительные — уточнить
```
