# Предпочтения по поиску (для судьи)

Свободный текст. Судья читает этот файл перед каждым отчётом и учитывает
твои вкусы и решения по конкретным вариантам. Предпочтения имеют приоритет
над общими критериями скилла, но НЕ над жёсткими правилами фильтра
(площадь/высота/локация — их меняют в `config.yaml`).

Правила разметки — просто пиши строками, судья разберёт:

- **Вето** («не предлагать …») — вариант не показывается в топе, а в конце
  отчёта одна строка «скрыто по предпочтениям: N».
- **Статус по варианту** (для блока «На контроле»): пиши id объявления и что с
  ним:
  - `watch 175` — наблюдаю, держи в блоке «На контроле»;
  - `позвонил 175: высота 7м в чистоте, торг возможен` — записал итог звонка;
  - `смотрю 175` — еду смотреть вживую;
  - `финалист 175` — кандидат в финал;
  - `отпало 175: дорого` — убрать из наблюдения и из топа.
- **Общие вкусы** — приоритеты и запреты словами.
